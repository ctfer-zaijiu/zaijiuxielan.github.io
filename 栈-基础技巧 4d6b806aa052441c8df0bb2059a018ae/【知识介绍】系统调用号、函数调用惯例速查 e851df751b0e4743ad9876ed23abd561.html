二、系统调用传参

x64系统

前三个：rdi、rsi、rdx

x32系统

前三个：ebx，ecx，edx

常见的系统调用传参

int execve(const char *文件路径, char *const argv[], char *const envp[]);

int open(const char *文件路径)

ssize_t read(int 从哪里读, void *buf, size_t nbyte);

ssize_t write(int 写到哪里, const void *buf, size_t nbyte);

其中：

stdin (标准输入)：文件描述符为 0
stdout (标准输出)：文件描述符为 1
stderr (标准错误输出)：文件描述符为 2

三、函数调用惯例

1-x64传参

寄存器传参，前6个rdi、rsi、rdx，cx，r8，r9第七个以后入栈传参入栈（call时会入栈PC）

2-x32传参

call前依次把函数传参从右往左顺序入栈，call时入栈PC

3-cedcl外平栈

函数名前面加下划线修饰

平栈方式：在call语句后面有add esp语句

4-stdcall内平栈

函数名修饰：下划线+函数名+@+参数的字节数

平栈方式：在最后一句，会有ret 8(硬编码：C2 08 00)，这句话相当于ret;add esp,8;
