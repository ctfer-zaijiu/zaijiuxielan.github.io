    （1）克隆 glibc 代码库：首先将 glibc 代码库克隆到你的本地机器上（如果你还没有它的源代码）：

    git clone https://github.com/bminor/glibc.git

    （2）切换到特定版本：进入 glibc 代码库目录，并切换到你所使用的特定版本（例如，glibc 2.27）：

    cd glibc
git checkout glibc-2.27

    （3）使用代码搜索工具在源代码文件中查找函数定义。

    使用 grep：

    grep -r 'dl_runtime_resolve' .

    使用 cscope（安装 cscope 并创建数据库）：

    sudo apt-get install cscope
find . -name "*.c" -o -name "*.h" > cscope.files
cscope -bq
