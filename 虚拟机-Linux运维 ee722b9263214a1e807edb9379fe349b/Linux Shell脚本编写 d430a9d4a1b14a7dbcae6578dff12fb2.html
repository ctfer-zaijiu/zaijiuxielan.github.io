三、循环

这里以“计算1+2+3+4+5+6+7+8+…+100的和，并输出到屏幕上”为例

1-for循环

#!/bin/bash

sum=0

for ((i=1; i<=100; i++));
do
    sum=$((sum + i))
done

echo "Sum using for loop: $sum"

循环条件：

for file in "$@"：这部分使用for循环遍历命令行参数（"$@" 表示所有的命令行参数）。

2-while循环

#!/bin/bash

sum=0
counter=1

while [ $counter -le 100 ]
do
    sum=$((sum + counter))
    counter=$((counter + 1))
done

echo "Sum using while loop: $sum"

五、文件读写

#!/bin/bash

# 指定用户列表文件路径
user_list_file="usernamelist.txt"

# 指定保存用户名与密码的文件路径
output_file="userpass.txt"

# 判断用户列表文件是否存在
if [ ! -f "$user_list_file" ]; then
    echo "Error: User list file '$user_list_file' not found."
    exit 1
fi

# 读取user_list_file指向的文件，创建用户并设置随机密码，保存到output_file指向的文件
while IFS= read -r username
do
    # 跳过空行
    if [ -z "$username" ]; then
        continue
    fi

    password=$(openssl rand -base64 6)

    # 创建用户
    sudo useradd -m -s /bin/bash "$username"

    # 设置用户密码
    echo "$username:$password" | sudo chpasswd

    # 保存用户名与密码到文件
    echo "$username:$password" >> "$output_file"
done < "$user_list_file"

echo "User accounts created successfully. Usernames and passwords saved to $output_file."
