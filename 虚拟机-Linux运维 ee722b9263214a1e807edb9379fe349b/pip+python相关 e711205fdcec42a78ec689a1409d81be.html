pip+python相关

建议：不要使用python2的工具，因为有些时候下载python2库会遇到各种各样麻烦的报错

配置pip

（1）升级pip3：

python -m pip install --upgrade pip

（2）暂时指定pip3使用的镜像：

pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn 库名   #使用清华源作为镜像

（3）pip3永久换源：

1.备份当前的pip配置文件：cp ~/.pip/pip.conf ~/.pip/pip.conf.bak

如果告诉你在当前用户目录下找不到pip目录，可以输入
mkdir -p ~/.pip

2.创建一个新的pip配置文件：
touch ~/.pip/pip.conf
3.使用编辑器打开新的pip配置文件：
vim ~/.pip/pip.conf
4.将以下内容复制到文件中：
[global]
index-url =
https://mirrors.aliyun.com/pypi/simple/
[install]

trusted-host=mirrors.aliyun.com
5.保存文件并退出编辑器。
现在，每次使用pip3时都会使用阿里云的镜像源。

如果你想恢复到默认的pip源，请使用以下命令删除pip配置文件：
rm ~/.pip/pip.conf
并将备份的配置文件重命名为原始文件名：
mv ~/.pip/pip.conf.bak ~/.pip/pip.conf

（4）管理多个版本的python对应的pip

  • {1}安装pip2、pip3

    用下面两行命令来安装pip包：

    curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python get-pip.py
    如果你要装pip2.7，则下载地址应该改为：https://bootstrap.pypa.io/pip/2.7/get-pip.py

    如果你要为Python3.6安装pip，就把上面第二行命令换成：

    python3.6 get-pip.py
  • {2}使用pip2、pip3

    这里我们pip2对应的是zaijiu用户的pip指令、pip3对应的是pip3指令

    pip2：

    然后我们zaijiu的pip2会抽风，它下载的库会默认下载到python3的库目录下(/usr/local/lib/python3.10/dist-packages)

    所以我们下载好后，还需要输入以下指令来换位：

    cp /usr/local/lib/python3.10/dist-packages/库文件.py /usr/local/lib/python2.7/dist-packages/库文件.py

    pip3

    • 使用不同版本的pip3

      pip命令对应的是一个在/usr/bin/里的同名脚本文件“pip”，或者是/home/zaijiu/.local/bin/里的同名脚本文件“pip”

      其内容如下：

      #!/usr/bin/python3.6

# -*- coding: utf-8 -*-
import re
import sys

from pip._internal import main

if __name__ == '__main__':
    sys.argv[0] = re.sub(r'(-script\.pyw?|\.exe)?$', '', sys.argv[0])
    sys.exit(main())

      这里第一行就是对pip脚本文件会检索到哪个pip版本的声明，如果我们想要它检索到python3.8的pip，就将其声明改为：

      #!/usr/bin/python3.8

一些路径信息

在kali里面，python路径和pip安装库路径是不一样的！

1-python库一般下在哪里

在我的kali2022环境下

python2的库被下载到：

/usr/local/lib/python2.7/dist-packages
这里很奇怪，我的zaiju用户好像对这个文件没有写权限，这里要chmod给他加一下

python3的库被下载到：

/usr/local/lib/python3.10/dist-packages

此外，也可以通过pip show <package_name>命令来查看已安装库的位置，如：

pip show requests
