一、相关基础

1-基础介绍

镜像：

os基础环境的各个数据文件和程序文件，基于层技术

容器：

进程和虚拟机环境，是状态机

卷：

容器产生的持久化存储的数据(也就是说将应用进程和持久化数据分离)，被单独地存储在卷上

卷不会因为容器停止或删除而丢失，卷可以轻易从容器中备份出来，也可以轻易地加入某个容器

2-docker项目结构

dockerfile：

是一个用于自动化构建 Docker 镜像的文本文件，类似于makefile。

二、docker应用

官方使用教程：https://docs.docker.com/manuals/

1-安装

这里是先安装docker引擎，他就相当于是核心解释器，有了它就可以跑容器了。然后再安装docker桌面，这一步是让我们可以再图形化界面下更方便操作。

这里只建议在kali上装docker，因为其他系统会出bug(痛，太痛辣~)

【通过apt安装】

  • kali通过apt安装docker引擎
    前提条件：

    1、需要主机支持虚拟化

    2、需要满足系统要求，参考官方文档：https://docs.docker.com/engine/install/ubuntu/#prerequisites

    1、安装docker的apt存储库文件

    # Add Docker's official GPG key（gpg密钥其实就是docker官网的公钥，存放为/etc/apt/keyrings/docker.asc文件）
sudo apt-get update
sudo apt-get install ca-certificates curl
sudo install -m 0755 -d /etc/apt/keyrings
sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg -o /etc/apt/keyrings/docker.asc
sudo chmod a+r /etc/apt/keyrings/docker.asc

# Add the repository to Apt sources（以下指令就是，从环境变量获取当前os架构，携带架构参数和公钥去请求docker官网下载对应的apt存储库文件，存为/etc/apt/sources.list.d/docker.list文件）
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.asc] https://download.docker.com/linux/ubuntu \
  $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
sudo apt-get update
    docker的apt存储库文件是docker官方提供的一个列表，其中列举了当前系统中为了装docker对应需要安装和维护的哪些程序和依赖

    2、安装 Docker 软件包

     sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
    以上是“安装最新版本”

    3、验证“是否安装成功”

     sudo docker run hello-world
    这会自动下载并运行官网库里面的一个专门用来验证docker是否安装成功的“hello world”镜像

  • kali安装docker引擎的可选项

    1、(可选)kali设置允许“在没有 root 权限的情况下运行 Docker”

    参考链接：https://docs.docker.com/engine/install/linux-postinstall/

    2、设置docker自启动(可选)

    sudo systemctl start docker
sudo systemctl enable docker
    取消docker自启动：
     sudo systemctl disable docker.service
 sudo systemctl disable containerd.service

    3、(可选)设置docker自带的“日志记录程序json-file”的相关配置

    参考链接：https://docs.docker.com/engine/logging/drivers/json-file/

  • kali安装docker桌面

    1、安装docker的apt存储库文件

    操作方式和“安装docker引擎”一样

    2、从官网下载软件包

    地址：https://docs.docker.com/desktop/install/linux/ubuntu/

其他安装方式：

1、从官网下载软件包安装之，链接：https://docs.docker.com/engine/install/ubuntu/#install-from-a-package

2、使用一次性脚本安装之，链接：https://docs.docker.com/engine/install/ubuntu/#install-using-the-convenience-script

【卸载】

  • kali卸载docker引擎

    1、卸载 Docker Engine、CLI、containerd 和 Docker Compose 软件包：

    sudo apt-get purge docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin docker-ce-rootless-extras
    注意，删除docker引擎时默认不会删除所有本地的“镜像、容器和卷”，如果要删除需要用以下命令手动删之

    2、(谨慎！)删除所有镜像、容器和卷

    sudo rm -rf /var/lib/docker
sudo rm -rf /var/lib/containerd
