进程管理

top开出一个类似“任务管理器”的实时显示界面(可以看出一些保活的木马)

而ps则是拍摄当前的进程快照然后输出

一、top指令

1-指令介绍

提供了一个动态的、实时更新的视图，显示：

  • 系统的进程+进程信息
  • 资源使用情况(如CPU和内存使用率)
其实现实际上就是每隔一段时间拍摄一个当前的进程快照然后输出

2-相关选项

我们先输入一个top，然后会进入交互界面，在交互界面可以输入以下这些指令：

【显示】

  • i：一个开关式命令。启用后，top将不显示那些状态为闲置（idle）或僵死（zombie）的进程。
  • f 或 F：从当前显示中添加或者删除项目。这允许用户自定义top显示的列。
  • m：在输出的表头显示物理内存和交换空间的使用情况。

【排序】

  • o 或 O：自定义改变显示项目的顺序。
  • P：根据CPU使用百分比大小进行排序，以查找最占用CPU资源的进程。
  • M：根据驻留内存大小进行排序，以找出哪些进程占用了最多的物理内存。
  • T：根据时间/累计时间进行排序，以找出运行时间最长的进程。

【进程管理】

  • k：终止一个进程。系统将提示用户输入需要终止的进程ID（PID）和发送的信号类型，默认信号是SIGTERM（15），用于请求进程终止。
  • r：重新安排一个进程的优先级别。系统将提示用户输入进程ID和新的nice值，用于调整进程的优先级。

【其他】

  • h：显示帮助画面，给出一些简短的命令总结说明。
  • q：退出程序。这是从top界面退出回到命令行的标准方式。
  • w：将当前设置写入~/.toprc文件中。这可以使top在下次启动时自动应用。

二、ps命令

1-指令介绍

拍摄快照，显示系统中当前运行的所有进程的信息。

2-相关选项

在ps命令可以中携带以下传参：

【常用】

-a、-e：显示所有进程

-u：以用户为主的格式来显示程序状况

-x：显示所有程序，不以终端机来区分

-f：用ASCII字符显示树状结构，表达程序间的相互关系

【其他】

-U <用户uid>：显示指定用户的进程

-t <终端机编号>：指定终端机编号，并列出属于该终端机的程序的状况

3-常用组合指令

ps -aux：以用户为主的格式显示所有进程的详细信息1。

ps -ef：显示所有进程的详细信息，包括进程的UID、PID、PPID、C、STIME、TTY、TIME和CMD
