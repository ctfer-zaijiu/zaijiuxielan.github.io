vim

1、相关指令

光标移动：

:数字    快速移动到第n行

ctrl+f    往下翻页(forward)

ctrl+b    往上翻页(backward)

增删复制：

dd    删除一整行

y数字y    复制包括当前光标所在的n行内容

p 粘贴

这些操作是不需要先打“:”符号且无回显的

查询、替换：

/str    文本字符串str搜索，可以按下 n 键来跳转到下一个匹配项

:%s/str1/str2/g    把全文中所有str1替换为str2

其中，%表示对全文做操作，s表示替换，g表示全局替换

撤销：

u    撤销 上一次操作

2、安装插件

【安装vim插件管理器】

gayhub地址：

https://github.com/junegunn/vim-plug

相关安装指令：

curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

【使用vim插件管理器】

在vim的配置文件（~/.vimrc）中写入以下内容：

call plug#begin('~/.vim/plugged')
Plug 'junegunn/vim-easy-align'  -- 可以这么写，其对应的是https://github.com/junegunn/vim-easy-align
Plug 'https://github.com/junegunn/vim-github-dashboard.git'  -- 也可以这么写，其对应的就是这一url
call plug#end()

然后打开vim：

使用:PlugInstall来下载这些指定的插件

使用:PlugStatus来查看当前这些插件的状态

使用:PlugUpdate来更新插件

如果更新了插件表在里面删除了一些插件，需要再使用:PlugClean来使之做删除的更新

【代码补全插件平台-coc.nvim】

安装：

装配其所依赖的Node.js环境：curl -sL install-node.now.sh/lts | bash

然后在~/.vimrc中写入：Plug 'neoclide/coc.nvim', {'branch': 'release'}

安装完后，在vim命令行中输入:CocInfo如果能够看到相关信息表示安装成功

配置对应语言的补全插件：

通过在vim内的命令模式输入:CocConfig来配置coc.nvim的配置文件coc-settings.json

{
"languageserver": {
    "clangd": {
      "command": "clangd",
      "rootPatterns": ["compile_flags.txt", "compile_commands.json"],
      "filetypes": ["c", "cc", "cpp", "c++", "objc", "objcpp"]
    }
  }
}
这种配置模式下，coc主要是依赖clangd进行代码自动化补全，所以需要事先安装好clangd

相关安装教程如下：https://zhuanlan.zhihu.com/p/566506467
